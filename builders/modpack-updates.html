Installation Methods

Method 1: Complete Pack Installation

Use this method for: New installations, major version updates, or troubleshooting corrupted installations.

  1. Download: Complete modpack ZIP file (approximately 450MB)
  2. Extract: Use 7-Zip or WinRAR to extract to a new folder
  3. Import: In your launcher (Prism recommended), select "Import Instance"
  4. Configure: Set Java version to 21 and allocate 6-10GB RAM
  5. Launch: Start the instance and verify SHAFTT mod is present

Method 2: Incremental Update Installation

Use this method for: Existing v1.1 installations, minor updates, or when bandwidth is limited.

  1. Backup: Copy your current instance folder to a safe location
  2. Download: Incremental update package
  3. Extract: Extract the ZIP file to a temporary location
  4. Navigate: Open your existing TrekCraft instance folder
  5. Update Mods: Copy new JAR files to mods/ folder, replace existing files
  6. Update Resources: Copy updated resource pack files to resourcepacks/
  7. Update Configs: Copy updated configuration files to config/
  8. Verify: Launch instance and check mod versions in Mod Menu
