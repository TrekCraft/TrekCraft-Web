Installing the TrekCraft Modpack

Versions

Minecraft Java Edition 1.21.1

Fabric Loader 0.16.9

Download Pack

The TrekCraft modpack comes with everything you need to explore and enjoy visiting the TrekCraft server. The pack is also fully comptabible with the Axiom Editor which is recommended for players who want to try creating their own sci-fi megabuilds. Axiom, and other helpful mods are linked in the Optional Mod section below.

Downloading this pack also constitutes an acceptance of the license terms as outlined here.

Installation Instructions

Step 1: Install Prism [Optional, but STRONGLY recommended]

Prism is our recommended, and only supported launcher for TrekCraft 2.0. This is for a few reasons, but is mainly because other launchers do not offer all of the tools needed to run our pack in the best way possible.

  • Download Prism here for your system
  • Follow the on-screen setup guide
  • Prism has great help guides for setup, and moving from other launchers here

‚ùóIf you decide to skip this step you'll be on your own for setting up, and maintaining your pack‚ùó

Step 2: Download and install the TrekCraft Modpack

Scroll to the section below for a link to download the pack!

  • Once you've downloaded the pack, you can simply drag & drop the .ZIP on-top of Prism and it should automatically import your instance.
  • If this fails, you can try adding it manually by opening Prism > Add Instance > Import from Zip > Select the downloaded .ZIP > Import

Step 3: Setup your Instance

  • Right click on the newly added instance > Edit > Settings
  • Check Java Installation, then click Auto-detect and select Java 21. You can download Java directly in Prism from this same menu if you don't already have it installed.
  • Check Memory and set the maximum to between 6000mb and 10000mb depending on your system.

Now you can click Launch and the game should begin to boot!

Step 4: First-Time Running the Game

Normally the game should boot up right away, and you'll see the TrekCraft menu screen. If your game crashes review the common fixes below. Otherwise you can skip to Step 5.

  • If your game fails to open due to Code -1 or Code 0 this is usually because Java 21 is not enabled. Double check you selected the proper Java in Step 2 and then re-try.
  • If you get an error with Packwiz like Failed to download mods.toml then there may be a server issue, or a poor internet connection. Either wait and try again later, or disable Packwiz here and manually download the mods.

Step 5: Personalize your Client!

Out of the box the client is setup to be well optimized for building really big ships in Minecraft, but everyone has their own preferences. Be sure to click around and change your settings to best suit you! Also make sure to enable the TrekCraft resource pack! before you hop on server, otherwise things might look strange üòÖ

FAQ

Below are some common issues, and compatiblity errors that come up when running the TrekCraft Modpack.

I want to disable a mod, but it keeps turning itself back on!?

This is Packwiz, our pack updater putting the mod back. To fix this go into Prism, right click on the instance, select Edit, go to the Settings tab, and then Custom Commands sub-tab. Un-check the checkbox and this will turn off Packwiz at launch. Re-checking this box will re-enable Packwiz.

Can I install the modpack with another launcher?

Yes you can, but we won't be able to help you if something goes wrong. The 1.21 setup page on our site has a list of each mod along with links to download them. You will also not receive automatic updates or optimizations. It's strongly recommended to use the Prism Launcher for TrekCraft, along with our official mod pack.

How can I move around the server easier?

In-game you can press "M" by default to open your map. You can also visit map.trekcraft.org for a full live map of the server. /gamemode spectator is also great for flying through the ships with ease.

Everything is really dark and hard to see

Pressing "G" by default will enable gammabright, which we use often for building.

KNOWN ISSUES

  • MultiMC is not updating to current versions of Fabric properly, and thus the TrekCraft pack will fail to launch despite appearing normal. We recommend all users move to Prism Launcher which is based on MultiMC, but is more actively developed and maintained.
  • Apple Silicon (M1/2/3/4) based devices do not support specific graphics API's used by modern Minecraft. Specifically connected textures (Connectedness), and smooth fonts (Caxton) do not run properly, and may cause crashes or text not rendering. Removing these mods will usually fix any issues.
  • Older GPU's, like NVIDIA 900 series cards likewise do not support modern API's used to render connected textures or smooth fonts. Disabling the Connectedness, and Caxton mods may be required to launch the pack.
  • Systems with less ram (16GB or less), or with older GPU's without much VRAM may find errors enabling the resource pack resource reload failed error's are common. Some users can fix this by giving their game more RAM, but this may not always work.
  • Running the client on a HDD is NOT recommended and may result in errors, crashes, or very long load times. Running TrekCraft on an SSD is strongly recommended.
